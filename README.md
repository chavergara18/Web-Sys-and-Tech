Simple Portfolio Tracker

A beginner-friendly web application that helps users track their financial investments. This project demonstrates key web development concepts including HTML structure, CSS styling, JavaScript interaction, and the integration of external stylesheets such as Normalize.css and Bootstrap.

Features

Add assets with quantity and price

Automatically calculates total portfolio value

Ability to clear/reset entries

Fully responsive layout using Bootstrap

Includes Normalize.css for consistent browser styling

Simple, clean UI designed for learning and demonstration

Technologies Used
Technology	Purpose
HTML5	Provides the structure of the webpage
CSS3	Custom styling for fonts, layout, and visuals
Normalize.css	Ensures consistent rendering across browsers
Bootstrap 5	Responsive layout, spacing, and UI components
JavaScript (Vanilla)	Handles portfolio logic, user inputs, and dynamic updates
